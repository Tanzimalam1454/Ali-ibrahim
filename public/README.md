# Public Folder - Add Your Images Here

This is where all your website images go.

## 📂 Folder Structure

```
public/
├── images/
│   ├── logo/              ← Add company logo here
│   │   └── logo.png       ← Your main logo
│   └── clients/           ← Add client logos here
│       ├── client-1.png
│       ├── client-2.png
│       └── client-3.png
```

## 🏢 1. Company Logo

### Where to add:
**`public/images/logo/logo.png`**

### Your Logo:
- Navy blue building outline (#2D4059)
- Red "AK" letters (#D84339)
- Arabic text at bottom

### Requirements:
- PNG with transparent background
- Minimum 500px width
- Will appear in header and footer

## 👥 2. Client Logos

### Where to add:
**`public/images/clients/`**

### File naming:
- `saudi-aramco.png`
- `sabic.png`
- `maaden.png`
- etc.

### Requirements:
- PNG with transparent background
- 200-300px width
- Lowercase names with hyphens

## ✅ Quick Start

1. **Save your company logo:**
   ```
   Drop your logo file here:
   public/images/logo/logo.png
   ```

2. **Save client logos:**
   ```
   Drop client logos here:
   public/images/clients/company-name.png
   ```

3. **Refresh website** - Done!

---

**That's it!** Just drop your images in these folders.
