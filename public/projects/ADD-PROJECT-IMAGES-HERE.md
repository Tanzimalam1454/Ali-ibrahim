# Project Images

Add your project photos to this folder with the following filenames:

## Required Images:

1. **king-salman-park.jpg** - King Salman Park (Riyadh)
2. **neom-hospital.jpg** - NEOM Hospital (NEOM City)
3. **aramco-stadium.jpg** - Aramco Stadium (Dhahran)
4. **ithra-cultural-center.jpg** - Ithra Cultural Center (Dhahran)
5. **chamber-of-commerce-tower.jpg** - Chamber of Commerce Tower (Riyadh)
6. **snt-gulf-factory.jpg** - SNT Gulf Factory (Industrial City)

## Image Requirements:

- **Format**: JPG or PNG
- **Recommended dimensions**: 800px × 600px (or similar 4:3 ratio)
- **File size**: Keep under 500KB each for optimal web performance
- **Quality**: High-quality professional photos showcasing the projects

## Tips:

- Use landscape-oriented images
- Ensure images are well-lit and show the project clearly
- Compress images for web before uploading
- Use consistent aspect ratios across all images for a uniform look

Once you add the images with these exact filenames, they will automatically appear on the website!
