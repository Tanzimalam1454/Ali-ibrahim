# Assets Folder Structure

This folder contains all images, logos, and media files for the Ali Ibrahim Al Khyat website.

## 📁 Folder Organization

### `/logos/company/`
**Company Brand Logos**
- Place your main company logo here
- Recommended files:
  - `logo.png` - Full color logo (current: navy & coral)
  - `logo-white.png` - White version for dark backgrounds
  - `logo-icon.png` - Icon/favicon version (just the AK mark)

**Requirements:**
- Format: PNG with transparent background
- Size: At least 500px width for header
- Color: Navy (#2D4059) outline, Coral (#D84339) AK letters

### `/logos/clients/`
**Client & Partner Logos**
- Add logos of companies you've worked with
- Name files descriptively: `client-name.png`

**Recommended naming:**
- `saudi-aramco.png`
- `sabic.png`
- `maaden.png`
- `sec.png`
- etc.

**Requirements:**
- Format: PNG with transparent background
- Size: 200-300px width recommended
- Grayscale or color versions accepted

### `/images/`
**General Website Images**
- Project photos
- Team photos
- Facility images
- Equipment photos

**Recommended structure:**
- `/images/projects/` - Project completion photos
- `/images/team/` - Team and workforce photos
- `/images/equipment/` - Fleet and equipment photos

## 🎨 Image Guidelines

### Logo Specifications
- **File format:** PNG (transparent background) or SVG
- **Resolution:** Minimum 2x for retina displays
- **Color space:** RGB
- **Compression:** Optimized for web

### Photo Guidelines
- **Format:** WebP (preferred) or JPG
- **Max width:** 1920px
- **Quality:** 80-85%
- **Optimization:** Use tools like TinyPNG or ImageOptim

## 📝 Adding Your Logos

1. **Company Logo:**
   ```
   Save your logo as: assets/logos/company/logo.png
   The website will automatically display it in the header and footer
   ```

2. **Client Logos:**
   ```
   1. Save client logos in: assets/logos/clients/
   2. Name format: company-name.png
   3. The website will display them in the "Clients" section
   ```

## ⚡ Quick Start

1. Place your company logo in `assets/logos/company/logo.png`
2. Add client logos to `assets/logos/clients/`
3. Refresh the website - logos will appear automatically!
