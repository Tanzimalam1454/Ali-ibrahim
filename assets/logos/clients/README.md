## File Naming Convention

Use lowercase with hyphens:
- `saudi-aramco.png`
- `sabic.png`
- `maaden.png`
- `sec.png`
- `binladin-group.png`

## Logo Requirements

- **Format:** PNG with transparent background
- **Size:** 200-300px width
- **Color:** Grayscale or original colors
- **Background:** Transparent
- **Quality:** High resolution (2x for retina)

## Adding Logos

1. Get official logo files from client websites or brand guidelines
2. Remove backgrounds (use remove.bg or Photoshop)
3. Resize to ~250px width
4. Save as PNG with transparency
5. Place in this folder
6. Refresh website to see changes
