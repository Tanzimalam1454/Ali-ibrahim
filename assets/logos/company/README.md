## ✅ File Requirements

### Primary Logo (`logo.png`)
- **Format:** PNG with transparent background
- **Dimensions:** Minimum 500px width
- **Resolution:** 2x for retina (1000px+ recommended)
- **Color Mode:** RGB
- **File Size:** Under 200KB (optimized)

### Optional: White Version (`logo-white.png`)
- For dark backgrounds (footer uses white version)
- Same dimensions as primary
- All elements in white

### Optional: Icon Only (`logo-icon.png`)
- Just the "AK" mark without building
- Square format (500x500px)
- For favicon or small displays

2. **Footer** - Bottom section (80px height, white version)
